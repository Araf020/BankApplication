This project is a simple bank application using Java.

## Features
* Create Account
* Deposit Money
* Withdraw Money
* Search Account
* Delete Account
* Display  Account Details
* Update Account

## How to run
* Clone the project
* Open the project in any IDE
* Run the BankApplication.java file
